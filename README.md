# 物业工作照片归档助手

## V2.2.5 运行日志与问题反馈中心

V2.2.5 将数据维护中心的“试运行问题记录”升级为“运行日志与问题反馈中心”，在保留手动问题反馈的同时，自动记录扫描、归档、台账读取、资料包导出、系统设置、整改闭环等关键异常。运行日志独立保存到本地 `trial-issues.json`，兼容旧试运行问题记录，不写入照片归档台账，也不会删除、移动、压缩或修改任何照片。

运行日志支持按记录来源、级别、状态、页面和关键词筛选，可查看技术详情、标记已处理或已忽略、删除单条记录、清理已处理记录，并可导出当前筛选结果为 Excel 或 CSV。技术详情会做长度限制和重复记录去重，避免日志文件被短时间重复异常刷屏。

## V2.2.4 照片处理入口与归档状态统一版

V2.2.4 将“照片分拣工作台”统一为物业日常照片处理主入口：日常杂图、多事项、多位置照片使用标准分拣模式；同一事项、同一分类、同一位置的一批照片，可在分拣工作台内部切换到快速归档模式统一套用归档信息。快速归档完成后，当前会话中的对应照片会同步标记为“已归档”或“归档失败”，避免在标准分拣模式中误以为仍未处理。

系统设置的“基础数据”只保留项目、部门、照片来源、水印分类与工作内容、照片阶段、处理状态、关键词和常见场景管理；配置备份、导入导出和恢复默认统一在“设置备份与恢复”操作。新增配置会置顶并自动聚焦，关键词支持粘贴文本批量导入，常见场景建议由本地内置规则生成，不使用 OCR 或 AI，也不会覆盖用户已经手工修改的内容。

## V2.2.2 试运行问题记录版

V2.2.2 在“数据维护中心”新增“试运行问题记录”，用于记录软件实际试用期间发现的问题、影响程度、处理状态和修复备注。记录支持按页面、类型、影响程度、处理状态组合筛选，也可搜索问题描述和处理备注，并将当前筛选结果另存为 Excel 或 CSV。

试运行记录独立保存在本地 `trial-issues.json`，不会写入照片归档台账，也不会删除、移动、压缩或修改照片、整改事项、配置和资料包。数据文件损坏时系统会停止读写并提示先备份、再检查文件，避免用空数据覆盖原记录。

安全边界：删除前必须成功备份 `照片归档台账.xlsx`；文件删除仅允许处理当前归档根目录内、被选中台账记录明确指向的单个图片文件。系统不会删除原始照片、照片来源目录、整个归档目录、未选中文件、资料包、整改数据或台账备份。

- 首页严格只读，不会删除、移动、压缩或修改照片，不会改写归档台账、整改事项或配置。

## V1.9.0 资料汇总中心基础版

V1.9.0 新增“资料汇总中心”，用于把已经归档的照片台账和本地整改闭环事项汇总成可核对、可复制、可导出的统计资料。该中心只做读取、筛选、统计和另存为新的 Excel 汇总台账，不会修改原始照片、归档台账或整改事项数据。

基础用法：
- 在左侧导航进入“资料汇总中心”。
- 选择或沿用当前归档根目录，点击“加载汇总数据”读取照片归档台账。
- 页面会同时尝试只读读取本地整改事项数据；如果没有整改数据，也不会影响照片汇总。
- 可按项目、部门、水印分类、工作内容、日期范围、关键词以及“是否纳入整改事项”筛选。
- 汇总区提供照片记录、文件存在/缺失、项目数、部门数、分类数、整改事项数等指标。
- 可查看分类汇总、项目汇总、部门汇总、整改汇总，以及照片明细和整改明细。
- 点击“复制汇总摘要”可复制一段纯文本统计摘要。
- 点击“导出汇总台账”可另存为独立 Excel 文件，包含汇总总览、分类汇总、项目汇总、部门汇总、整改汇总、照片明细和整改明细。

安全边界：
- 不删除、不移动、不压缩任何原始照片或归档照片。
- 不修改、不覆盖原始 Excel 归档台账。
- 不修改整改闭环运行时数据；如存在 `rectification/rectification-items.json`，仅做只读兼容读取。
- 不做 OCR、AI 识图、云同步、Word/PDF 汇报导出。

## V1.8.0 整改闭环中心基础版
V1.8.0 新增“整改闭环中心”，用于把已归档照片进一步转化为可跟踪、可核对、可导出的整改事项。

基础用法：
- 在左侧导航进入“整改闭环中心”，可新建整改事项，也可从已有归档记录中选择照片创建整改事项。
- 新建事项需填写项目、责任部门、问题位置、问题标题、问题描述、整改要求和截止日期；水印分类、工作内容、处理人、关键词、备注和整改前照片可按需补充。

- 关联照片分为整改前、整改中、整改后。关联时只记录照片路径，不复制、不移动、不删除、不压缩照片。
- 详情区可打开关联照片、打开照片所在文件夹、复制整改事项摘要。
- 顶部可导出当前筛选结果为独立的“整改闭环台账”Excel，不会修改原始归档台账。

安全边界：整改闭环中心只维护本地整改事项数据，不会删除、移动、压缩或修改原始照片，也不会改写原始归档 Excel 台账。

V1.3.0 新增“照片分拣工作台”，用于处理同一照片文件夹中包含多类事项、多批次归档信息的真实场景。该页面采用工作台式布局：顶部是照片来源目录与主工具栏，左侧是状态筛选与分组列表，中间是照片缩略图网格，右侧是照片详情、归档信息表单、辅助填写和主操作按钮，底部显示全量状态统计。

基础能力包括：

- 缩略图网格 / 列表视图查看照片，支持缩略图小、中、大切换。
- 支持单选、Ctrl 多选、Shift 连续选择、全选当前页、选择未分拣、反选当前页和取消选择。
- 支持为选中照片批量应用归档信息，每张照片独立保存归档信息。
- 支持清除选中照片归档信息，状态回到“未分拣”，不会删除原始照片。
- 支持状态筛选、文件名搜索和基础分页浏览。
- 支持常见场景套用、关键词推荐和备注模板联动。
- 支持保存 / 加载分拣草稿，草稿只保存照片路径、状态和归档信息，不复制照片。
- 支持只为“已分拣”照片生成分拣归档预览，未分拣、已忽略和未生成预览照片不会归档。
- 支持归档前确认弹窗，确认后只复制已生成预览照片并追加 Excel 台账。

安全规则保持不变：原始照片只读取和复制，不移动、不删除、不压缩；清空列表、重新扫描、清除分拣信息都不会修改原图。归档文件管理、归档副本删除、台账删除或标记删除等能力不属于 V1.3.0 基础版范围，后续单独规划。

这是一个面向物业公司日常办公的本地桌面照片归档工具，用于把马克水印相机、手机相册、微信导出的工作照片，按项目、部门、水印分类、工作内容、位置/区域、事项名称、日期和照片阶段标准化归档，并自动生成 Excel 照片归档台账。

## V1.2.0 配置管理中心

V1.2.0 新增“配置管理中心”，可在首页点击“配置管理”，也可通过顶部菜单“设置 / 配置管理中心”打开。配置管理中心支持维护项目、部门、照片来源、水印分类、工作内容、照片阶段、处理状态、关键词和常见场景。

配置采用“双层配置”机制：

```text
程序内置默认配置：开发环境 config/，打包环境 resources/config

```

首次启动或用户配置缺失时，系统会自动从内置默认配置复制一份到用户配置目录。之后所有新增、修改、停用、删除、排序、导入和恢复操作都只写入用户配置目录，不直接修改 `resources/config`。软件升级不会覆盖用户自定义配置。

### 新增水印分类

打开“配置管理中心 / 水印分类与工作内容”，左侧是水印分类列表。点击“新增”，填写分类名称、分类说明、是否兜底和兜底提示。保存后主界面会立即刷新，水印分类下拉框会显示启用状态的新分类。

### 新增工作内容

打开“配置管理中心 / 水印分类与工作内容”，先在左侧选中某个水印分类，再在右侧“该分类下的工作内容”中点击“新增工作内容”。工作内容必须归属于一个水印分类，不能脱离分类单独存在。可填写工作内容名称、说明、推荐关键词和备注模板。保存后回到首页，选择对应水印分类即可看到新增工作内容。

在“工作内容”中填写“推荐关键词”和“备注模板”。首页选择该工作内容后，关键词推荐会优先包含这些关键词；备注模板区域也会出现对应模板，点击后可填入备注栏并继续手动修改。

### 停用分类和工作内容

停用水印分类后，该分类不会出现在首页水印分类下拉框中，其下工作内容也不会出现在归档表单中。停用某个工作内容后，只会影响该分类下的该项内容，首页选择该分类时不再显示该工作内容。停用不会修改历史台账，也不会影响已经归档的照片文件。

### 导出、导入和恢复默认

打开“配置管理中心 / 配置备份/导入导出”：

- “导出全部配置”会把当前用户配置导出为一个 JSON 备份文件。
- “导入配置”会选择 JSON 备份文件，导入前自动备份当前配置，导入后刷新主界面。
- “恢复默认配置”会从程序内置默认配置恢复，恢复前自动备份当前用户配置。
- 每次保存配置前都会自动备份当前配置，系统保留最近 30 个备份。

### 照片分拣工作台与快速归档模式

照片分拣工作台是物业日常照片处理主入口，适合每天从相机、手机或微信导出的杂图整理。标准分拣模式支持逐张选择照片、批量套用归档信息、生成分拣归档预览并确认归档。

快速归档模式位于照片分拣工作台内部，适合同一事项、同一分类、同一位置的一批照片统一套用归档信息。快速归档模式不用于逐张勾选不同事项照片；如果照片来源中混有多个事项、多个位置或多个部门，请优先使用标准分拣模式。

快速归档模式继续保留选择照片文件夹、选择归档根目录、路径记忆、最近目录、归档信息表单、常见场景快速套用、关键词推荐、照片扫描、归档预览、确认归档和 Excel 台账追加。快速归档成功后，当前会话内对应照片会在标准分拣模式中显示“已归档”；归档失败的照片显示“归档失败”。

## 归档流程

1. 选择照片文件夹。
2. 选择归档根目录。
3. 点击“扫描照片”，在左侧照片列表核对照片。
4. 在右侧填写项目、部门、水印分类、工作内容、位置/区域、事项名称等归档信息。
5. 点击“生成归档预览”，左侧会自动切换到归档预览，检查新文件名和归档摘要。
6. 必要时在预览表格中逐张修改照片阶段、关键词、备注。
7. 点击“确认归档”，在弹窗中核对归档信息、安全说明和分页说明。
8. 点击弹窗中的“确认归档”，系统复制照片并追加写入 Excel 台账；点击“返回修改”则关闭弹窗，不执行归档。

系统会根据水印分类和工作内容提供适合物业现场留痕的备注模板，例如楼道杂物、飞线充电、消防通道违停、设备房巡检、维修处理、时间地点水印等。点击模板后会填入备注栏，后续可继续手动编辑。

### 最近使用记录

归档成功后，系统会在本地浏览器存储中保存最近 10 条归档表单组合，包括项目、部门、照片来源、水印分类、工作内容、位置/区域、事项名称、照片阶段、处理状态、关键词和备注。用户可一键套用，也可清空最近记录。该记录只保存在本机，不上传云端。

## 字段规则

“工作内容”是标准工作项，来自当前水印分类下的配置，仍为必填。

“事项名称”用于描述本次照片对应的具体事项，不再强制必填；如果为空，生成预览、文件名、归档路径和 Excel 台账会自动使用“工作内容”作为兜底事项名称。

“位置/区域”建议填写，便于后期检索和迎检复核；如果为空，生成预览、文件名、归档路径和 Excel 台账会自动使用“现场”作为兜底位置。系统不提供最近位置、常用位置或一键套用位置，避免高频变化位置被误选。

## 水印分类

水印分类沿用马克水印相机常见物业工作分类：安全管理类、电动摩托车违规管理、机动车违规管理、工程类专用、巡查检查类、客服工作类、绿化保洁类、会议/培训/宣传/外联、时间地点水印。

选择水印分类后，工作内容下拉框会自动切换到对应内容。

## 文件夹规则

归档根目录 / 项目 / 年份 / 月份 / 部门 / 水印分类 / 工作内容 / 日期_位置区域_事项名称 / 照片阶段 /

## 文件名规则

日期_项目_水印分类_工作内容_位置区域_事项名称_照片阶段_三位序号.原扩展名

文件名会自动清理 Windows 非法字符；目标文件已存在时会自动追加序号，避免覆盖。

## 台账字段

日期、项目、部门、照片来源、水印分类、工作内容、位置/区域、事项名称、照片阶段、处理状态、新文件名、原文件名、关键词、备注、归档路径、归档时间。为兼容旧台账，已有 Excel 表头可能仍显示“具体位置/工作事项”，但写入值会使用 V1.2.5 的兜底后的“位置/区域/事项名称”。

## 安全说明

本工具默认只复制照片，不移动、不删除原图，不修改原图内容，不压缩画质。归档前必须生成预览并由用户确认。

V1.0.1 已验证连续归档同一批照片时不会覆盖已有目标文件，会自动追加序号；Excel 台账采用追加写入，不会清空旧数据。

V1.2.5 起，归档前仍会检查照片文件夹、归档根目录、照片来源、项目、部门、水印分类、工作内容、日期和照片阶段等必填项；事项名称为空时使用工作内容兜底，位置/区域为空时使用“现场”兜底。

V1.2.6 起，点击确认归档后会先显示归档前确认弹窗，提示本次归档数量、归档根目录、原始照片保护、分页只影响查看以及 Excel 台账追加规则。只有在弹窗中再次点击“确认归档”才会真正复制照片。

V1.0.2 增强了真实试运行中的提示和保护：长文件名和长目录段会自动截断，避免 Windows 路径过长影响归档。

## V1.5.0 归档资料包导出中心基础版
V1.5.0 在“归档记录”页新增“生成资料包”能力，用于把当前筛选结果或已勾选的归档记录整理成一个可提交、可检查、可归档的资料包。

- 默认使用当前筛选结果生成资料包；如果在结果表格中勾选了记录，则优先只使用已勾选记录。
- 生成前会要求选择资料包保存位置，并弹出确认窗口，显示记录数量、文件存在/缺失数量、目标目录、资料包名称、分组规则和安全说明。
- 资料包内部包含 `01_照片资料`、`02_资料目录` 和 `资料包说明.txt`。
- `01_照片资料` 会按“项目 / 水印分类 / 工作内容”建立子目录并复制归档照片副本；字段为空时使用“未分类”。
- `02_资料目录/资料包目录.xlsx` 只记录本次导出的资料，包含原归档文件路径、资料包内相对路径、文件状态和导出结果。
- 文件缺失或复制失败会写入资料包目录 Excel，不会修改原始归档台账。
- 资料包导出只复制照片，不移动、不删除、不压缩原始照片或归档照片，也不会修改原台账。
- 生成完成后可直接打开资料包或打开资料目录 Excel 所在文件夹。

# V1.6.0 系统设置与基础数据管理中心基础版

V1.6.0 将“系统设置”扩展为统一的本地管理中心，用于集中维护基础数据、默认目录、资料包导出偏好、设置备份恢复和系统运行信息。该页面只写入本地配置与 settings.json，不会删除、移动、压缩照片，也不会修改已有 Excel 台账。

主要能力：

- 基础数据：复用配置管理能力，统一维护项目、部门、照片来源、水印分类、工作内容、关键词、照片阶段、处理状态和常见场景。
- 默认目录：支持设置默认照片导入目录、默认归档根目录、默认资料包导出目录，并保留最近目录和上次目录记忆开关。

- 设置中心仍遵守只复制、不移动、不删除、不压缩原图的安全规则。
# V1.7.0 数据维护中心基础版

V1.7.0 新增“数据维护中心”，用于只读检查本地配置、默认目录、归档台账、分拣进度草稿和资料包导出目录的健康状态。

数据维护中心可以帮助确认：

- 系统配置文件是否存在、是否可读取，基础数据数量是否正常。
- 默认照片导入目录、默认归档根目录、默认资料包导出目录是否已配置且可访问。
- 当前归档根目录下是否存在照片归档台账，台账记录数、文件存在数和文件缺失数是否正常。
- 本地分拣进度草稿目录是否存在，是否有较早未处理的草稿。
- 默认资料包导出目录下是否存在疑似资料包结构。
- 根据检查结果给出维护建议，并支持跳转到系统设置、归档记录或照片分拣工作台继续处理。

安全说明：数据维护中心仅做只读检查，只提供刷新检查、打开目录、复制路径和跳转相关页面，不会删除、移动、压缩或修改任何照片、台账、配置、分拣进度或资料包。
