# 物业工作照片归档助手

这是一个面向物业公司日常办公的本地桌面照片归档工具，用于把马克水印相机、手机相册、微信导出的工作照片，按项目、部门、水印分类、工作内容、具体位置、日期和照片阶段标准化归档，并自动生成 Excel 照片归档台账。

## 适用场景

- 日常巡查、工程维修、秩序管理、环境保洁、客服沟通等工作照片留痕。
- 迎检资料、整改闭环、月度汇报、现场取证材料整理。
- 多来源照片二次整理归档。

## 不做什么

- 不拍照，不替代马克水印相机。
- 不做团队管理、云同步、考勤管理、权限系统。
- V1.0 不做 AI 识图、OCR 水印识别。
- 不删除、不移动、不压缩原始照片。

## 安装与运行

```bash
npm install
npm run dev
```

如果 Windows PowerShell 提示禁止运行 `npm.ps1`，可以改用：

```bash
npm.cmd install
npm.cmd run dev
```

本项目会把 Electron 运行数据写入项目内的 `.runtime` 目录，避免默认写入系统 AppData 目录导致权限问题。

打包前可执行：

```bash
npm run build
```

## 归档流程

1. 选择照片文件夹。
2. 选择归档根目录。
3. 填写项目、部门、水印分类、工作内容、位置、事项等归档信息。
4. 点击“扫描照片”。
5. 点击“生成归档预览”，检查新文件名和目标路径。
6. 必要时在预览表格中逐张修改照片阶段、关键词、备注。
7. 点击“确认归档”，系统复制照片并追加写入 Excel 台账。

## V1.1.0 关键词增强版

V1.1.0 面向物业办公人员的高频录入场景，新增“常见场景快速套用”“关键词推荐”“备注模板”“最近使用记录”，减少重复填写。

### 常见场景快速套用

首页提供常见场景卡片，当前默认包含楼道杂物清理、飞线充电治理、消防通道违停、电动车乱停乱放、闭门器维修、公共照明维修、公共设施设备维修、设备房巡检、消防设施巡检、环境卫生巡查、垃圾清运、绿化修剪、消杀作业、资料整理归档、小区宣传通知、时间地点水印现场确认。

点击场景后会自动填充水印分类、工作内容、工作事项建议、处理状态建议、照片阶段建议、推荐关键词和备注模板。用户仍可继续手动修改。

### 关键词推荐

系统会根据水印分类、工作内容、具体位置、工作事项、处理状态和常见场景综合推荐短关键词。点击推荐关键词可加入关键词输入框，再次点击可取消；也可以一键清空关键词。

### 备注模板

系统会根据水印分类和工作内容提供适合物业现场留痕的备注模板，例如楼道杂物、飞线充电、消防通道违停、设备房巡检、维修处理、时间地点水印等。点击模板后会填入备注栏，后续可继续手动编辑。

### 最近使用记录

归档成功后，系统会在本地浏览器存储中保存最近 10 条归档表单组合，包括项目、部门、照片来源、水印分类、工作内容、具体位置、工作事项、照片阶段、处理状态、关键词和备注。用户可一键套用，也可清空最近记录。该记录只保存在本机，不上传云端。

## 水印分类

水印分类沿用马克水印相机常见物业工作分类：安全管理类、电动摩托车违规管理、机动车违规管理、工程类专用、巡查检查类、客服工作类、绿化保洁类、会议/培训/宣传/外联、时间地点水印。

选择水印分类后，工作内容下拉框会自动切换到对应内容。

## 文件夹规则

归档根目录 / 项目 / 年份 / 月份 / 部门 / 水印分类 / 工作内容 / 日期_具体位置_工作事项 / 照片阶段 /

## 文件名规则

日期_项目_水印分类_工作内容_具体位置_工作事项_照片阶段_三位序号.原扩展名

文件名会自动清理 Windows 非法字符；目标文件已存在时会自动追加序号，避免覆盖。

## 台账字段

日期、项目、部门、照片来源、水印分类、工作内容、具体位置、工作事项、照片阶段、处理状态、新文件名、原文件名、关键词、备注、归档路径、归档时间。

## 安全说明

本工具默认只复制照片，不移动、不删除原图，不修改原图内容，不压缩画质。归档前必须生成预览并由用户确认。

V1.0.1 已验证连续归档同一批照片时不会覆盖已有目标文件，会自动追加序号；Excel 台账采用追加写入，不会清空旧数据。

V1.0.2 增强了真实试运行中的提示和保护：归档前会检查照片文件夹、归档根目录、项目、部门、水印分类、工作内容、日期、具体位置、工作事项、照片阶段等必填项；时间地点水印会额外提醒填写备注；长文件名和长目录段会自动截断，避免 Windows 路径过长影响归档。
