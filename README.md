# 物业工作照片归档助手

这是一个面向物业公司日常办公的本地桌面照片归档工具，用于把马克水印相机、手机相册、微信导出的工作照片，按项目、部门、水印分类、工作内容、位置/区域、事项名称、日期和照片阶段标准化归档，并自动生成 Excel 照片归档台账。

## 适用场景

- 日常巡查、工程维修、秩序管理、环境保洁、客服沟通等工作照片留痕。
- 迎检资料、整改闭环、月度汇报、现场取证材料整理。
- 多来源照片二次整理归档。

## 不做什么

- 不拍照，不替代马克水印相机。
- 不做团队管理、云同步、考勤管理、权限系统。
- V1.0 不做 AI 识图、OCR 水印识别。
- 不删除、不移动、不压缩原始照片。

## 安装与运行

```bash
npm install
npm run dev
```

如果 Windows PowerShell 提示禁止运行 `npm.ps1`，可以改用：

```bash
npm.cmd install
npm.cmd run dev
```

开发环境会把 Electron 运行数据写入项目内的 `.runtime` 目录；打包后会写入“用户文档/物业工作照片归档助手/.runtime”，避免写入程序安装目录或系统 AppData 目录导致权限问题。

打包前可执行：

```bash
npm run build
```

## Windows 打包

V1.1.1 起支持使用 electron-builder 打包 Windows 可执行程序：

```bash
npm run dist
```

打包产物默认输出到 `release/`：

- `release/物业工作照片归档助手-1.2.0-portable.exe`：便携版，可直接运行。
- `release/物业工作照片归档助手-1.2.0-setup.exe`：NSIS 安装包。
- `release/win-unpacked/物业工作照片归档助手.exe`：未压缩调试版。

打包后内置默认配置会随程序放入 `resources/config`，用户修改后的配置不会写回程序目录，而是保存到“用户文档/物业工作照片归档助手/config”。照片台账不会写入程序安装目录，仍写入用户选择的归档根目录。

如果便携版在受限电脑上无法启动，请确认系统临时目录 `TEMP/TMP` 可写；程序运行缓存会写入“用户文档/物业工作照片归档助手/.runtime”，避免写入安装目录。

### 打包后白屏排查

V1.1.1 修复版已将 Vite 构建 `base` 设置为 `./`，确保 Electron `file://` 环境下资源以相对路径加载。构建后的 `dist/index.html` 应包含类似 `./assets/index-xxxx.js` 和 `./assets/index-xxxx.css` 的引用，不能是 `/assets/...` 或 `/C:/assets/...`。

打包版已设置中文应用菜单：文件、编辑、视图、窗口、帮助。

## 路径记忆与默认目录

V1.1.2 起，应用会自动记住上次选择的照片文件夹和归档根目录。下次启动时，如果目录仍然存在，会自动恢复；如果目录不存在，会在顶部状态栏提示“上次目录不存在，请重新选择”，不会崩溃。

路径设置保存在本机：

```text
用户文档/物业工作照片归档助手/settings.json
```

保存内容包括上次照片文件夹、上次归档根目录、默认归档根目录、最近照片文件夹、最近归档根目录。最近目录最多各保留 5 条，只保存本地路径，不保存照片内容，不上传云端。

归档根目录第一次选择后会自动作为默认归档根目录；也可以在界面中点击“设为默认归档根目录”手动更新。

## V1.2.0 配置管理中心

V1.2.0 新增“配置管理中心”，可在首页点击“配置管理”，也可通过顶部菜单“设置 / 配置管理中心”打开。配置管理中心支持维护项目、部门、照片来源、水印分类、工作内容、照片阶段、处理状态、关键词和常见场景。

配置采用“双层配置”机制：

```text
程序内置默认配置：开发环境 config/，打包环境 resources/config
用户自定义配置：用户文档/物业工作照片归档助手/config
配置自动备份：用户文档/物业工作照片归档助手/config-backup
路径设置文件：用户文档/物业工作照片归档助手/settings.json
```

首次启动或用户配置缺失时，系统会自动从内置默认配置复制一份到用户配置目录。之后所有新增、修改、停用、删除、排序、导入和恢复操作都只写入用户配置目录，不直接修改 `resources/config`。软件升级不会覆盖用户自定义配置。

### 新增水印分类

打开“配置管理中心 / 水印分类与工作内容”，左侧是水印分类列表。点击“新增”，填写分类名称、分类说明、是否兜底和兜底提示。保存后主界面会立即刷新，水印分类下拉框会显示启用状态的新分类。

### 新增工作内容

打开“配置管理中心 / 水印分类与工作内容”，先在左侧选中某个水印分类，再在右侧“该分类下的工作内容”中点击“新增工作内容”。工作内容必须归属于一个水印分类，不能脱离分类单独存在。可填写工作内容名称、说明、推荐关键词和备注模板。保存后回到首页，选择对应水印分类即可看到新增工作内容。

### 推荐关键词和备注模板

在“工作内容”中填写“推荐关键词”和“备注模板”。首页选择该工作内容后，关键词推荐会优先包含这些关键词；备注模板区域也会出现对应模板，点击后可填入备注栏并继续手动修改。

### 停用分类和工作内容

停用水印分类后，该分类不会出现在首页水印分类下拉框中，其下工作内容也不会出现在归档表单中。停用某个工作内容后，只会影响该分类下的该项内容，首页选择该分类时不再显示该工作内容。停用不会修改历史台账，也不会影响已经归档的照片文件。

### 导出、导入和恢复默认

打开“配置管理中心 / 配置备份/导入导出”：

- “导出全部配置”会把当前用户配置导出为一个 JSON 备份文件。
- “导入配置”会选择 JSON 备份文件，导入前自动备份当前配置，导入后刷新主界面。
- “恢复默认配置”会从程序内置默认配置恢复，恢复前自动备份当前用户配置。
- 每次保存配置前都会自动备份当前配置，系统保留最近 30 个备份。

## 归档流程

1. 选择照片文件夹。
2. 选择归档根目录。
3. 点击“扫描照片”，在左侧照片列表核对照片。
4. 在右侧填写项目、部门、水印分类、工作内容、位置/区域、事项名称等归档信息。
5. 点击“生成归档预览”，左侧会自动切换到归档预览，检查新文件名和归档摘要。
6. 必要时在预览表格中逐张修改照片阶段、关键词、备注。
7. 点击“确认归档”，系统复制照片并追加写入 Excel 台账。

## V1.1.0 关键词增强版

V1.1.0 面向物业办公人员的高频录入场景，新增“常见场景快速套用”“关键词推荐”“备注模板”“最近使用记录”，减少重复填写。

默认项目配置为：潇湘新区二期、香辰康园、其他。

### 常见场景快速套用

首页提供常见场景卡片，当前默认包含楼道杂物清理、飞线充电治理、消防通道违停、电动车乱停乱放、闭门器维修、公共照明维修、公共设施设备维修、设备房巡检、消防设施巡检、环境卫生巡查、垃圾清运、绿化修剪、消杀作业、资料整理归档、小区宣传通知、时间地点水印现场确认。

点击场景后会自动填充水印分类、工作内容、事项名称建议、处理状态建议、照片阶段建议、推荐关键词和备注模板。位置/区域只提供提示，不会强制自动填入，用户仍可继续手动修改。

### 关键词推荐

系统会根据水印分类、工作内容、位置/区域、事项名称、处理状态和常见场景综合推荐短关键词。点击推荐关键词可加入关键词输入框，再次点击可取消；也可以一键清空关键词。

### 备注模板

系统会根据水印分类和工作内容提供适合物业现场留痕的备注模板，例如楼道杂物、飞线充电、消防通道违停、设备房巡检、维修处理、时间地点水印等。点击模板后会填入备注栏，后续可继续手动编辑。

### 最近使用记录

归档成功后，系统会在本地浏览器存储中保存最近 10 条归档表单组合，包括项目、部门、照片来源、水印分类、工作内容、位置/区域、事项名称、照片阶段、处理状态、关键词和备注。用户可一键套用，也可清空最近记录。该记录只保存在本机，不上传云端。

## 字段规则

“工作内容”是标准工作项，来自当前水印分类下的配置，仍为必填。

“事项名称”用于描述本次照片对应的具体事项，不再强制必填；如果为空，生成预览、文件名、归档路径和 Excel 台账会自动使用“工作内容”作为兜底事项名称。

“位置/区域”建议填写，便于后期检索和迎检复核；如果为空，生成预览、文件名、归档路径和 Excel 台账会自动使用“现场”作为兜底位置。系统不提供最近位置、常用位置或一键套用位置，避免高频变化位置被误选。

## 水印分类

水印分类沿用马克水印相机常见物业工作分类：安全管理类、电动摩托车违规管理、机动车违规管理、工程类专用、巡查检查类、客服工作类、绿化保洁类、会议/培训/宣传/外联、时间地点水印。

选择水印分类后，工作内容下拉框会自动切换到对应内容。

## 文件夹规则

归档根目录 / 项目 / 年份 / 月份 / 部门 / 水印分类 / 工作内容 / 日期_位置区域_事项名称 / 照片阶段 /

## 文件名规则

日期_项目_水印分类_工作内容_位置区域_事项名称_照片阶段_三位序号.原扩展名

文件名会自动清理 Windows 非法字符；目标文件已存在时会自动追加序号，避免覆盖。

## 台账字段

日期、项目、部门、照片来源、水印分类、工作内容、位置/区域、事项名称、照片阶段、处理状态、新文件名、原文件名、关键词、备注、归档路径、归档时间。为兼容旧台账，已有 Excel 表头可能仍显示“具体位置/工作事项”，但写入值会使用 V1.2.5 的兜底后的“位置/区域/事项名称”。

## 安全说明

本工具默认只复制照片，不移动、不删除原图，不修改原图内容，不压缩画质。归档前必须生成预览并由用户确认。

V1.0.1 已验证连续归档同一批照片时不会覆盖已有目标文件，会自动追加序号；Excel 台账采用追加写入，不会清空旧数据。

V1.2.5 起，归档前仍会检查照片文件夹、归档根目录、照片来源、项目、部门、水印分类、工作内容、日期和照片阶段等必填项；事项名称为空时使用工作内容兜底，位置/区域为空时使用“现场”兜底。

V1.0.2 增强了真实试运行中的提示和保护：长文件名和长目录段会自动截断，避免 Windows 路径过长影响归档。
