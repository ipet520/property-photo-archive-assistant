# 更新记录

## V1.0.1 运行验证与稳定性修复版

- 完成 `npm install`、`npm run dev`、`npm run build` 运行验证。
- 将 Electron 固定到 31.7.7，提升当前 Windows 环境下的启动稳定性。
- 将 Electron 主进程、preload 和服务层切换为 CommonJS 入口，减少 Electron 与 ESM 加载兼容风险。
- 将 Electron 用户数据、日志、崩溃转储、运行缓存改为项目内 `.runtime` 目录，避免写入系统 AppData 时权限受限。
- 补充 `.npm-cache`、`.electron-cache`、`.runtime` 等本地运行目录忽略规则。
- 验证照片扫描、归档预览、只复制不移动原图、目标重名自动避让、Excel 台账追加写入。

## V1.0.0 基础归档版

- 初始化 Electron + React + Vite 项目。
- 完成照片文件夹选择与图片扫描。
- 完成水印分类与工作内容联动配置。
- 完成中文归档表单。
- 完成归档预览表格。
- 完成确认后复制归档。
- 完成照片自动重命名。
- 完成 Excel 照片归档台账生成与追加写入。
